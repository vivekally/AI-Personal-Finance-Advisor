# AI Personal Finance Advisor

A personal finance operating system for newcomers to Canada — the same guidance a $400/hr advisor would give, for $20/month.

## What This Is

An onboarding wizard that takes a newcomer immigrant from "I have savings and no idea what to do" to a personalized RRSP/TFSA/FHSA plan.

**Target users:** Immigrants to Canada in their first 1–3 years. Employed, $50k–$200k in savings, unfamiliar with the Canadian financial system.
